exercise 4.1:
linear gap penalty = 2
insertion length = 5
total cost = 5 * 2 = 10

exercise 4.2:
affine gap penalty: W(k) = gamma + delta * k
gamma = 8
delta = 0.5
k = 5
W(5) = 8 + 0.5 * 5 = 8 + 2.5 = 10.5

exercise 4.3:
the affine model is biologically more realistic because a 5-bp insertion is usually one single mutational event, 
not five independent 1-bp events. It penalizes opening a gap strongly, then extending the same gap more gently
this reflects biology better than the linear model, which treats each added gap position as equally costly

exercise 6.5.4:
research institutes keep large BLAST databases locally because searches are faster, sensitive data does not need to leave the institute, and BLAST can be integrated into automated pipelines
local copies also allow many researchers to run repeated jobs without depending on the public web interface or queue limits
